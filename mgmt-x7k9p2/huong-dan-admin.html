Hình 4.2: Giao diện upload hình ảnh

1
Nhấn nút Chọn ảnh hoặc Upload
2
Chọn file hình ảnh từ máy tính (JPG, PNG, WebP)
3

Giới hạn kích thước

Mỗi file hình ảnh không được vượt quá 2MB. Vui lòng nén hình trước khi upload nếu cần.

Xóa hình ảnh

  1. Nhấn nút Xóa bên cạnh hình cần xóa
  2. Xác nhận thao tác xóa
  3. Hệ thống sẽ khôi phục về hình ảnh mặc định

Xem hình phóng to

Nhấn vào hình preview để xem ở kích thước đầy đủ. Nhấn ra ngoài hoặc nút X để đóng.
